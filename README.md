# Project Overview
RetailFlow is a fictional mid-size e-commerce company. You have been brought on as the Data Engineer responsible for building their first cloud data platform. Over the course of this capstone you will take raw, messy order data all the way through a governed data lake, into two parallel analytics engines (Amazon Redshift and Databricks), and out to business-ready dashboards.

# Learning Objectives
* Write a well-organized Python/Boto3 script to upload files to S3, with basic logging and retry logic.
* Design a simple three-zone (Bronze/Silver/Gold) S3 data lake with sensible folder naming, basic security, and a lifecycle rule.
* Get a first hands-on look at access governance with AWS Lake Formation and LFTags.
* Build a beginner-friendly AWS Glue ETL job that cleans data, re-runs safely using Job Bookmarks, and checks data quality automatically.
* Model and load a simple star schema in Amazon Redshift Serverless, and see how Redshift Spectrum queries S3 data directly.
* Build a basic Delta Lake medallion pipeline on Databricks using Auto Loader and Delta Live Tables, governed by Unity Catalog.
* Present your results through a simple SQL dashboard on both engines.
